### Folder Purpose Summary
- **`actions/`** вҖ“ ServerвҖ‘side functions that run on Cloudflare Workers (R2) to handle QRвҖ‘related operations and file uploads.
- **`app/`** вҖ“ Next.js App Router structure. Contains both public pages (`(public)`) and authenticated sections (`(private)`). Also holds API routes, global CSS, and static assets.
- **`components/`** вҖ“ All React components, split into logical groups:
  - `block/` вҖ“ Larger UI blocks used across the app.
  - `custom/` вҖ“ ProjectвҖ‘specific components such as the phone scanner, navbar, and QR display.
  - `renderer/` вҖ“ Utilities for rendering dynamic pages and QR templates.
  - `ui/` вҖ“ Primitive UI primitives (buttons, dialogs, etc.) following the shadcnвҖ‘ui pattern.
  - `theme-provider.tsx` вҖ“ Provides dark/light theme context.
- **`config/`** вҖ“ Constant values and configuration helpers used throughout the codebase.
- **`db/`** вҖ“ Database schema definitions and helper functions for interacting with the data layer.
- **`hooks/`** вҖ“ ReвҖ‘usable React hooks.
- **`lib/`** вҖ“ Pure JavaScript/TypeScript utility libraries (auth helpers, validators, image upload, etc.).
- **`services/`** вҖ“ ServiceвҖ‘layer abstractions that wrap external APIs or internal endpoints.
- **`store/`** вҖ“ StateвҖ‘management stores (e.g., Zustand) for clientвҖ‘side state.
- **`styles/`** вҖ“ Global stylesheet modules (currently empty вҖ“ ready for future CSS/SCSS files).
- **`proxy.ts`** вҖ“ Entry point for the Cloudflare Workers proxy used by the app.

Feel free to adjust, add, or remove any entries as the project evolves. This structure now lists **every** folder and file under `src` without omission.
