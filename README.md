# QR‑Scan

> **Dynamic QR code generation, scanning, and management platform**


## Overview
QR‑Scan is a modern web application that empowers businesses and developers to **create**, **customize**, **track**, and **scan** dynamic QR codes in real‑time. Built with a premium UI/UX, the app showcases glass‑morphism, smooth micro‑animations, and a dark‑mode ready design that feels **state‑of‑the‑art**.

Key capabilities include:
- **Dynamic QR generation** with custom styles, colors, logos, and templates.
- **Real‑time analytics** – see scans, locations, and device data instantly.
- **QR‑code scanning** directly in the browser using a realistic phone‑frame component.

## Features
- **Dynamic QR Builder** – choose templates, colors, logos, and preview instantly.
- **QR Scanner** – realistic phone‑frame component (`PhoneScanner`) with smooth animation.
- **Analytics Dashboard** – private `(private)` routes for campaign insights.
- **Trusted By** – carousel of partner logos with hover effects.
- **Responsive Design** – mobile‑first layout, fluid grids, and adaptive scaling.
- **Dark / Light Mode** – automatic theme detection with a toggle.
- **Extensible Architecture** – plug‑in new components via the `src/components/` hierarchy.
